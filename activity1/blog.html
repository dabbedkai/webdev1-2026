My Web Dev Journey

Week 1: Surviving HTML

Published on June 9, 2026

This week was all about getting to know HTML. I learned about the structure of an HTML document, including the head and body sections. I also explored various HTML tags like headings, paragraphs, lists, and links. It was a bit overwhelming at first, but I'm excited to build my own web pages!

One of the most interesting things I learned was how to create links using the anchor tag. I can now link different pages together, which is essential for building a website. I'm looking forward to learning CSS next week to make my pages look more stylish!
